About

The Studio

Diamant Games is a casual mobile game studio based in Berlin, Germany. We focus on crafting casual mobile games that are simple to pick up, satisfying to play, and beautifully designed.

Our growing portfolio spans card games, puzzles, word games, and board games — each built with attention to detail and a love for the classics.
